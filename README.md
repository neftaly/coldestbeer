# Coldest Beer

ESP32-C6 BLE monitor for dual JK-BMS batteries and Alpicool fridge, with e-paper display.

## BOM

- Seeed XIAO ESP32-C6
- WeAct 2.13" e-paper display (3-color, red/black/white)
- DC-DC buck converter (48V+ input, 5V output, e.g. XL7015)
- 100k resistor (voltage divider, high side)
- 4.7k resistor (voltage divider, low side)
- External antenna + U.FL pigtail (optional, improves BLE range)
- Project box
